Fonoaudiólogo Educacional - Secretaria de Educação de Gramado (RS)

Este repositório contém o código-fonte de um aplicativo web interativo de página única (SPA) que sintetiza a pesquisa e a estrutura de atribuições para o cargo de Fonoaudiólogo Educacional na Secretaria Municipal de Educação de Gramado, Rio Grande do Sul.

Objetivo do Projeto

O objetivo principal é transformar a documentação oficial do cargo (Anexo de Atribuições) em uma ferramenta de consulta digital, intuitiva e acessível. A aplicação detalha o escopo de trabalho do profissional, focando na atuação consultiva, estratégica (Nível I - Promoção/Prevenção) e de gestão de fluxo (Nível II - Detecção/Articulação), conforme as diretrizes de Inclusão e o contexto intersetorial do município.

Estrutura e Tecnologia

O projeto é inteiramente contido em um único arquivo HTML (index.html) e utiliza tecnologias estáticas:

HTML5/JavaScript: Estrutura e lógica de interatividade.

Tailwind CSS (CDN): Estilização e layout responsivo.

Chart.js (CDN): Visualização de dados (Gráfico de Alocação de Foco).
